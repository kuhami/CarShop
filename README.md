# CarShop
汽车商城小程序，可自定义 作为‘商城’类小程序模版

### 项目说明：

- **基于微信小程序开发的一款移动端汽车商城**
- **基于 weui.wxss、es6 前端技术开发**
- **前台[CarShop](https://github.com/kuhami/CarShop/)**

### 目录结构
    
    CarShop/
      |-pages/                      # 小程序页面相关文件
          |- car_detail
            |- index.js
            |- index.json
            |- index.wxml
            |- index.wxss
          |- ...
      |-app.js                      # 小程序逻辑
      |-app.json                    # 小程序公共设置
      |-app.wxss                    # 小程序公共样式表
      |-...
